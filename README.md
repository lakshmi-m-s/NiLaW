# NiLaW

Wedding invitation website (React, Vite, TypeScript, Tailwind).

## Prerequisites

- **Node.js** 18 or newer (see `engines` in `package.json`). If you use [nvm](https://github.com/nvm-sh/nvm), run `nvm use` in this directory to match `.nvmrc`.

## Run locally

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the development server:

   ```bash
   npm run dev
   ```

3. Open the URL shown in the terminal (by default [http://localhost:5173/](http://localhost:5173/)).

## Production build

```bash
npm run build
```

Serve the `dist` folder with any static host, or run a local preview:

```bash
npm run preview
```
